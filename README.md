# esp8266_inHome_weather
esp8266 local web server with:
 - NodeMCU v1.0;
 - BME280 - temperature, humidity, pressure sensor;
 - MQ135 - air quality sensor;
 - LCD SSD1306;
 - RTC ds3231 - 'offline' I2C clock;
 - Bootstrap web framework;
